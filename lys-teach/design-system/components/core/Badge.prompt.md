Small rounded pill for statuses and categories — order state, "Available now", MCP tags.

```jsx
<Badge tone="accent">in_progress</Badge>
<Badge tone="success">Available now</Badge>
<Badge tone="warning">pending</Badge>
<Badge tone="neutral" outline>MCP</Badge>
```

Tones: `accent` (purple, default), `success`, `warning`, `danger`, `neutral`. `outline` drops the fill for a border-only chip.
